# helpdesk

## O que é?

Uma instalação base do framework PHP Laravel, configurada como um boilerplate (ponto de partida) para acelerar o desenvolvimento de novas aplicações.


How to run:

```bash
cd ./helpdesk && yarn dev
```

- Instalar Nuxt
- Instalar Laravel
- Preparar Laravel para trabalhar como API
- Preparar Laravel Migrations e Seeders: php artisan migrate:fresh --seed
