# duplicati

**Site Oficial:** [https://www.duplicati.com/](https://www.duplicati.com/)


## O que é?

O Duplicati é um cliente de backup gratuito que armazena arquivos de forma criptografada em serviços de nuvem online, como S3, Google Drive, entre outros.


- https://duplicati.com/
- https://hub.docker.com/r/linuxserver/duplicati

How to run:

```bash
cd ./duplicati && yarn dev
```
